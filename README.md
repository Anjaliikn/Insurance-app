# 📱 Insurance App (InsureEasy)

A mobile application built using **React Native (CLI) and TypeScript** that allows users to manage insurance policies, view details, and perform basic actions like signup, login, and navigation across features.

---

## 🚀 Features

* 🔐 User Authentication (Login & Signup UI)
* 🏠 Home Dashboard
* 📄 View Insurance Policies
* 📱 Clean and responsive UI
* ⚡ Smooth navigation between screens
* ✅ Form validation for signup & login

---

## 🛠️ Tech Stack

* React Native CLI
* TypeScript (TSX)
* React Navigation
* JavaScript (ES6+)

---

---

## ⚙️ Installation & Setup

1. Clone or extract the project

2. Install dependencies:

```
npm install
```

3. Run the app:

```
npm start
```

4. Run on device/emulator:

```
npx react-native run-android
```
